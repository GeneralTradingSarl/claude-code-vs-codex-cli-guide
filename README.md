# Claude Code vs Codex CLI: The Complete Guide

**Type less, code more.** A complete, verified reference guide to the two leading AI command-line coding agents: **Claude Code** (Anthropic) and **Codex CLI** (OpenAI). Every slash command, every CLI flag, every keyboard shortcut, plus a full head-to-head comparison to help you pick the right terminal AI agent for your workflow.

Available in [French (PDF)](Claude-Code-vs-Codex-CLI-Guide-FR.pdf) and [English (PDF)](Claude-Code-vs-Codex-CLI-Guide-EN.pdf).

## What's inside

- **Claude Code full command reference**: every slash command (`/code-review`, `/compact`, `/agents`, `/mcp`, `/plan`, and 40+ more), every `claude` CLI flag (`--permission-mode`, `--allowed-tools`, `--output-format`, `--worktree`, etc.), and every keyboard shortcut.
- **Claude Code configuration deep dive**: CLAUDE.md persistent memory, Skills (`SKILL.md`), Hooks (PreToolUse, PostToolUse, SessionStart...), MCP (Model Context Protocol) servers, Subagents, Plan Mode, and the Claude Agent SDK (Python & TypeScript).
- **Codex CLI full command reference**: every built-in slash command, every global flag (`--sandbox`, `--oss`, `--profile`, `--remote`...), every subcommand (`codex resume`, `codex apply`, `codex archive`...), and every interactive shortcut.
- **Codex CLI configuration**: AGENTS.md, `~/.codex/config.toml`, and the profiles system.
- **Side-by-side comparison table**: models, memory files, extensibility, sandboxing, remote sessions, SDKs, and local/open-source model support.
- **Quick cheat sheet**: the fastest commands for both tools on one page.
